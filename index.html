DISK OPERATING SYSTEM:
DOS is a abbreviation for Disk Operating System which was the first operating system used for personal comouter.
  1. DOS was developed in JULY 1980.
  2. DOS was developed of a 16-bits for their personal use only.
  3. It is a unique tasking operating system.
  4. The specific name of DOS is PC-DOS.
